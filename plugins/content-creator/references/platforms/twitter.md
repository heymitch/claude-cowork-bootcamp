# Twitter / X Platform Guide

## Format Overview

- **Single tweet:** 280 characters max. That's it.
- **Threads:** 5-15 tweets. Each tweet must stand alone as a complete thought.
- **Hook tweet:** The first tweet is everything. If it doesn't stop the scroll, nothing else gets read.
- **Tone:** Conversational. Like texting a smart friend, not writing an essay.
- **Hashtags:** Almost never. Hashtags feel like 2015. Only if you're targeting a specific community/event.
- **Emojis:** Only if the user's voice profile uses them. Never performative.

---

## Single Tweet Patterns

### The One-Liner

```
[One punchy sentence that contains a complete idea.]
```

Rules:
- Under 240 characters so it doesn't look cramped
- Should be quotable — something someone would retweet without context
- No "thread:" at the end unless it IS a thread

### The Mini-Thread Teaser

```
[Single tweet that teases your thread.] 🧵
```

Use this to tease a thread from a profile post or a different post. Points to the thread.

---

## Thread Structure

**Optimal length:** 5-10 tweets for most content. 10-15 for deep dives. More than 15 = you needed an article.

### Standard Thread (insight with evidence)

```
Tweet 1 (Hook): [Bold claim or surprising insight. End with "A thread:" or "Here's how:"]

Tweet 2 (Context): [Why does this matter? Set up the problem or the gap.]

Tweet 3-7 (Insight/Evidence): [One point per tweet. Specific. Concrete. No padding.]

Tweet N-1 (Takeaway): [What the reader should DO with this. Concrete action.]

Tweet N (CTA): [Invite engagement: "What did I miss?" / "Bookmark this for later." Optional: link to newsletter or resource.]
```

### Story Thread

```
Tweet 1 (Hook): [The most surprising or emotionally resonant moment. "I did X. Here's what happened:"]

Tweet 2 (Setup): [Who, what, where. The context before the event.]

Tweet 3-6 (Story): [Beat by beat. Each tweet is a scene or a moment.]

Tweet 7 (Turning point): [The moment everything changed.]

Tweet 8 (Lesson): [What you learned. Be direct.]

Tweet 9 (Takeaway for reader): [How does this lesson apply to THEM?]

Tweet 10 (CTA): [Question or link to longer piece.]
```

## Rules for Every Tweet

- **Each tweet stands alone.** If someone reads only tweet 3, they should still get value.
- **No fluff.** If a tweet is just connecting two other tweets ("So why does this matter?"), cut it or merge.
- **Vary tweet length.** Some one-liners. Some 3-4 sentences filling the 280 chars. Don't make them uniform.
- **Concrete over abstract.** "I got 10,000 impressions" beats "I saw good results."
- **No throat-clearing.** Don't start a thread with "I've been thinking a lot about X lately." Start with the thing you've been thinking about.
- **End threads with a question or a bookmark prompt.** "What did I miss?" or "Save this for later." Drives engagement without being sycophantic.

---

## Hook Tweet Patterns (What Makes People Click "Show More")

1. **The surprising result:** "I tested X for 30 days. The result shocked me."
2. **The contrarian take:** "Everyone says X. They're wrong."
3. **The numbered insight:** "7 things nobody tells you about [topic]:"
4. **The mistake reveal:** "I made a $10K mistake. Here's what I learned:"
5. **The bold claim:** "[Common belief] is killing your [outcome]."

**Rule:** The hook tweet should be the best tweet in the thread, not a placeholder. If the hook doesn't earn a click, nothing else matters.

---

## What NOT to Do

- Don't write threads longer than the content deserves. 3-tweet threads exist.
- Don't use "1/" at the start unless it's a proper thread (3+ tweets).
- Don't end every tweet with a period. Natural punctuation feels more human.
- Don't beg for engagement: "RT this to spread the word!"
- Don't write in corporate voice. Twitter rewards people, not brands.
- Don't pad with filler tweets just to hit a tweet count.
