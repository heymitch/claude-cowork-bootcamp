# Voice Mixer

## Flow

1. Accept text input (from archetype-analyzer or direct paste)
2. Rewrite the text in each of the 5 archetype styles:
   - **The Storyteller** — add dates, times, locations, personal examples
   - **The Opinionater** — inject conviction, emphasis, conviction adverbs
   - **The Fact Presenter** — add stats, research references, evidence
   - **The Frameworker** — transform into actionable steps, checklist, or process
   - **The F-Bomber** — add sarcasm, abrasive language, ranting tone
3. Present in a comparison table with original + 5 rewrites
4. Ask user which style(s) resonate or could strengthen the original

## Rules

- Keep length consistent across rewrites (within 10-15% of original)
- Do NOT change the core message or claim — only the voice/tone
- Each rewrite must be clearly identifiable as its archetype
- Preserve any links, citations, or factual claims from original
- Do NOT combine archetypes in a single rewrite (keep them pure)

## Output

**Voice Remix Table:**
| Style | Rewrite |
|-------|---------|
| Original | [Original text] |
| The Storyteller | [Rewritten with narrative elements] |
| The Opinionater | [Rewritten with conviction/emphasis] |
| The Fact Presenter | [Rewritten with stats/research] |
| The Frameworker | [Rewritten as actionable steps] |
| The F-Bomber | [Rewritten with sarcasm/edge] |

**Reflection Question:**
Which style(s) feel closest to your voice? Which could strengthen this piece?
